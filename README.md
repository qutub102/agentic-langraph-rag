# Agentic Document Assistant

An Agentic RAG system that ingests documents, semantically indexes them, and answers user questions using a LangGraph-driven retrieval and reasoning workflow.

## Features

- **Document Ingestion**: Upload PDF documents for processing
- **Semantic Indexing**: Automatic chunking, embedding, and vector storage in Qdrant
- **Intelligent Q&A**: LangGraph-powered agentic workflow for retrieval and reasoning
- **Citation Support**: Answers include citations with quotes from source documents
- **Real-time Status**: Live job status updates via polling
- **Modern UI**: Fully integrated React frontend with TypeScript
